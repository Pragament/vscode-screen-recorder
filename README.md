## Screen Recorder

Record Visual Studio Code Screen

## Install

- <kbd>Command</kbd> + <kbd>P</kbd>
- Type: `ext install screen-recorder`

## Support

- [x] macOS
- [ ] Windows
- [ ] Linux

## Usage

- Click `Record Screen` / `Stop Recording` in Status Bar.
